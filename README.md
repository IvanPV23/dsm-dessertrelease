# Proyecto Dessert Release - Tarea Académica

Este repositorio contiene el código del proyecto "Dessert Release", desarrollado como parte de un trabajo para el curso de Desarrollo de Sistemas Móviles.

## Origen del Código

El código y la estructura de este proyecto se basan en la ruta de aprendizaje [Android Basics with Compose](https://developer.android.com/courses/android-basics-compose/course) ofrecida por Google para desarrolladores de Android.
